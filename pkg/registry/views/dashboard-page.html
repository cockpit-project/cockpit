Docker commands

Log into the registry:

Your login credentials do not give you access to use the docker registry from the command line. 
$ sudo docker login -p {{config.registry.password}} -e unused -u unused {{config.registry.host}}

Push an image:

$ sudo docker tag myimage {{config.registry.host}}/project/name:tag
$ sudo docker push {{config.registry.host}}/project/name

Pull an image:

$ sudo docker pull {{config.registry.host}}/project/name:tag
